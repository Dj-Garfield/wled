# WLED
Einfaches WLED Board
 - Für 5V and 12V LED Streifen (unterschiedliche Bestückung möglich)
 - Beispiele für die Verbindung mit dem Netzteil/LED
 - Einfache Montage, Kein Kabelsalat, technisch saubere und sichere Ausführung
 - Tauschbare Komponenten
 - SW: https://github.com/Aircoookie/WLED

Aktuell verkaufe ich 2 Platinen, die ich übrig habe,   [auf ebay Kleinanzeigen](https://www.ebay-kleinanzeigen.de/s-anzeige/led-steuerung-mit-wled-sw-u-a-fuer-tv-ambilight-hyperion/1697329063-168-16390) (die 5V Version).
Habe gerade auch noch ein LED Streifen übrig (WS2812B, 5m, 30 LED/m): [auf ebay Kleinanzeigen](https://www.ebay-kleinanzeigen.de/s-anzeige/ws2812b-led-strip-5m-controller-mit-wled-sw-ambilight-/1697265323-168-16390).
